# Walleca 設計書

このファイルは設計全体の入口です。詳細は `docs/設計/` 配下へ分割しています。

## 1. 設計方針
- Web 優先の MVP とし、クライアントは Next.js App Router を採用する
- API 中心のクライアント分離構成を採用する
- OCR 画像は永続保存せず、確認前の一時データとしてのみ扱う
- レシート、サブスクともに OCR 結果はユーザー確認後に確定する
- 個人利用を前提とし、共有家計簿やグループ機能は対象外とする
